# Xiangyu Xu  InGen Dynamics Data Analyst Internship Portfolio

This repository contains my weekly deliverables for the InGen Dynamics Data Analyst Internship Program.

## Program Focus

The project focuses on synthetic data analysis for physical AI and robotics platforms, including exploratory data analysis, SQL fleet querying, regression modeling, predictive analytics, KPI design, and dashboard development.

## Important Note

This repository only uses synthetic or public-domain data. It does not include InGen Dynamics confidential information, internal documents, customer data, or proprietary materials.

## Folder Structure

- `data/`  synthetic datasets
- `notebooks/`  Jupyter notebooks for EDA, regression, prediction, and segmentation
- `reports/`  weekly reports and analysis briefs
- `sql/`  SQL database scripts and analytical queries
- `dashboard/`  dashboard files
- `capstone/`  final capstone report and executive deck
- `weekly/`  weekly data analyst logs
